# macdowsOS Tool UI 使用文档

-----------------------------------------------------------------------------------------
## 一 安装和运行
1.首先 从发布页下载 release.7z 或 release.zip 

2.随后在文件资源管理器中使用 Windows 的解压选项或第三方解压工具(这里建议使用 [7-zip](https://www.7-zip.org/))解压

3.随后在解压后的文件夹中找到macdowsOS Tool UI.exe 右键 点击 "以管理员身份运行" 

-----------------------------------------------------------------------------------------

##  二 使用
### 1) 注意事项
 1.请<span style="color:#e60000;">务必</span>以管理员身份运行，虽然macdowsOS Tool 不会检测当前运行权限，无管理员权限 macdowsOS Tool 仍然可运行，但是在 macdowsOS Tool 修改部分系统文件时<span style="color:#e60000;">可能</span>因为权限不够导致系统文件损坏的情况
 
 2.macdowsOS Tool 安装某些组件使用了自动操作技术，可能会被某些杀软识别为病毒，建议在安装前退出或禁用所有杀软
 
 3.作者不保证 macdowsOS Tool 不会导致系统出现任何问题，但是 macdowsOS Tool 的还原点组件因为存在严重的bug 现已停用，<span style="color:#e60000;">强烈建议</span>在安装前设置系统还原点并做好系统文件备份

## 三 问题排查
### 1) 安装前
#### <span style="color:#66a3e0;">1.在点击完免责申明等弹窗之后 macdowsOS Tool 主程序时出现以下或与以下类似的报错</span>
 
`无法启动此程序，因为计算机中丢失 XXXXX.dll。尝试重新安装该程序以解决此问题。`

`应用程序无法启动，因为应用程序的并行配置不正确。有关详细信息，请参阅应用程序事件日志，或使用命令行 sxstrace.exe 工具获取详细信息。`

`应用程序正常初始化失败。请单击“确定”，终止应用程序。`

`应用程序发生异常 未知的软件异常`

`异常代码 0xC0000005：ACCESS_VIOLATION 访问违例`

`该内存不能为 "read"。`

`该内存不能为 "written"。`
####  **解决方案：** 安装或修复Microsoft Visual C++ 可再发行程序包
[下载链接](https://learn.microsoft.com/zh-cn/cpp/windows/latest-supported-vc-redist?view=msvc-170)

#### <span style="color:#66a3e0;">2.在运行macdowsOS Tool时出现以下或与以下类似的报错</span>

`由于找不到 sfml-xxxx.dll，无法继续执行代码。重新安装程序可能会解决此问题。`

#### **解决方案：** macdowsOS Tool 图形依赖损坏或丢失，请重新下载完整的macdowsOS Tool 程序包

### 2) 安装时
#### <span style="color:#66a3e0;"> 1,安装时进度条长时间卡住(一般 3 分钟以上进度条不动)，且无任何报错或弹窗提示，点击窗口时程序无响应
</span>

#### **解决方案：** 将 macdowsOS Tool 目录中的 log.txt 反馈给我(或者你自己看的懂知道什么问题你自己解决也行)

#### <span style="color:#66a3e0;"> 2.出现其他任何报错
#### **解决方案：** 把报错截图发我我告诉你怎么办
  
### 3) 安装后
#### <span style="color:#66a3e0;"> 1,安装后文件资源管理器无限重启,在此过程中屏幕出现一行黑底白字的提示
#### **解决方案：**  出现此情况是因为macdowsOS Tool 安装的某些组件不支持你的系统 ，win+i 打开设置 卸载一个名为 StartAllBack 的程序即可。
  
#### <span style="color:#66a3e0;"> 2,安装后在登录界面闪屏/进入桌面后文件资源管理器无限重启(无提示)
#### **解决方案：**   出现此情况为 macdowsOS Tool 安装的主题不支持你的系统 ，win+i 打开设置，将主题更换为默认主题即可(若卡在登录界面的话这边建议使用还原点还原)
  
#### <span style="color:#66a3e0;">3.安装后所有图标变为黑色/透明
#### **解决方案：**  出现此情况为 macdowsOS Tool 在修改系统文件时操作中断或被阻止 导致系统文件损坏 使用 macdowsOS Tool 自带的系统修复的功能修复即可
  
#### <span style="color:#66a3e0;">4.安装后桌面文件被隐藏(这是正常的),但是我不习惯这样，如何恢复？
#### **解决方案：** 桌面右键-查看-显示桌面图标

#### <span style="color:#66a3e0;">5.安装后我想更换壁纸，但是过一会就会恢复为 macOS 壁纸怎么办？
#### **解决方案：** 右键 Dock 栏打开 MyDockFinder 偏好设置-壁纸 将壁纸设置为无 然后再设置你自己的壁纸即可。

#### <span style="color:#66a3e0;">6.Dock 栏/Finder 栏太大/太小怎么办？
#### **解决方案：** 右键 Dock 栏打开 MyDockFinder 偏好设置-通用-全局界面大小 这个是调 Finder 栏的 偏好设置-Dock-通用-缩放大小 这个是调 Dock 栏大小的
