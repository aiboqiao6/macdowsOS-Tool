## 📖简介
欢迎大家使用macdowsOStool

使用c++编写的自动化windows11仿macOS工具

原理上使用了SendMessage函数和其他模拟用户操作的方法实现了自动化操作

> macdowsOS Tool UI 安装功能高度依赖窗口/组件名称查找 请确保系统语言为 简体·中文 避免安装时因语言不同导致工具无法正常控制安装程序  目前暂时没有适配其他语言的计划

## 💻系统支持

>**✅支持的Windows版本** 
>
>Windows11 x64 (不支持 ARM64）
>
>Windows10 x64（1809及以上版本,不支持ARM64和x32）
>
>❌不支持某些精简过度的Windows(例如精简了DWM,Theme服务的系统)
