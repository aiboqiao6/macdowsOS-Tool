# macdowsOS Tool
## Introduction

Welcome to macdowsOStool.

An automated Windows 11 macOS-style tool written in C++.

In principle, it uses the SendMessage function and other methods of simulating user operations to achieve automated operations.

## System Support

>**Supported Windows versions**
>
>Windows 11 x64 (ARM64 not supported)
>
>Windows 10 x64 (Version 1809 and above; ARM64 and x32 not supported)
>
>Some overly stripped-down Windows versions are not supported (e.g, systems with DWM or Theme services removed).
